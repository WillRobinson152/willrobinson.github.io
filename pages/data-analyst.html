Data Analyst

As an editor at OPIS, a Dow Jones company, Robinson has completed a number of projects that help his team assess real-time, spot market prices for natural gas liquids and report on the NGL sector. These projects include:

  • Automated content generation
    Robinson created a Python module for describing price and inventory movements, which is used by multiple OPIS teams to automatically write alerts about:
    • weekly propane inventories and key statistics released by the U.S. Energy Information Administration
    • weekly long and short bets on derivatives for NGLs, crude and other energy products reported by the U.S. Commodities Futures Trading Commission
    • monthly propane and butane inventories reported by the Canada Energy Regulator
  • Data visualization
    Robinson created data visualizations in Tableau and Power BI, such as this visualization of propane inventories:
  • Reading data from screenshots Robinson created a desktop application using Python's Tkinter, Pytesseract and OpenPyxl that allows users to upload screenshots of market data and automatically convert it to a formatted spreadsheet.
  • Data query tools Robinson created a desktop application as a user interface for an internal OPIS API that employees can use to query historical price data. The interface allows users to click through options to add as query parameters.
    Robinson also automated recurring data queries to external APIs, such as retrieving certain propane statistics from the U.S. Energy Information adminsitration. View the code here.
